# SeedoMac v1.6.0 "Master Architecture" Edition

> **种子工作法 (Seedo Method) 的 macOS 终极实现** —— 专为深度工作者打造的生产力监控与健康平衡套件。

SeedoMac v1.6.0 引入了全新的“大师架构”，在数据完整性、视觉统一性以及由于重启带来的时间损耗方面实现了跨越式进化。

---

## ✨ 1.6.0 大师版核心更新

### 1. 永恒时钟 (Strict Persistence)
*   **秒级存档**: 计时器现在会实时存档至磁盘。即便软件意外退出、系统崩溃或强制重启，计时器也会在重新开启时精准恢复到退出前的秒数。
*   **物理续关**: 专注进度不再因外界干扰而中断，确保你的每一分钟努力都被严格锁定。

### 2. 联邦提醒生态 (Unified Reminder UI)
*   **全场景对齐**: 短时休息、长时休息、以及**中断检测（欢迎回来）**现在全屏统一采用同款精致 UI。
*   **视觉仪式感**: 背景虚化、色调动态切换（绿/橙/蓝），在所有状态切换节点提供一致的品牌体验。

### 3. 高保真双字段记录 (Dual-Input Logging)
*   **标题 + 备注**: 专注结束后，系统会引导你拆分填入“任务总结（标题）”与“即时笔记（过程）”。
*   **资产化数据**: 你的活动历史不再是模糊的碎片，而是条目清晰、细节详实的生产力资产。

### 4. 离线 DeepFocus
*   **物理隐私**: 进入 DeepFocus 时自动切断后台屏幕追踪引擎，实现真正的无干扰离线环境。

## 🚀 快速开始

### 系统要求
*   macOS 12.0 或更高版本。
*   需要授予 **辅助功能 (Accessibility)** 权限以实现窗口追踪功能。

### 安装
1.  下载最新的 `SeedoMac_v1.6.0.dmg`。
2.  将 SeedoMac 拖入“应用程序”文件夹。
3.  启动后，在系统偏好设置中授予权限。
